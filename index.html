About Solution

The PDF Summarization System is a scalable, cost-effective pipeline that combines extractive and abstractive summarization techniques to efficiently process PDF documents. The system extracts text using PyMuPDF, followed by spaCy for extractive summarization, reducing the document size by 70-90% without any external API or token costs. A low-cost LLM API (such as OpenAI’s ChatGPT4o) is then used for abstractive summarization, generating high-quality summaries at minimal cost. Token analysis is performed to track and optimize API usage, and summaries are stored in a SQL database for future retrieval. The solution minimizes LLM token usage, accelerates document processing with parallel workers, and routes only large or critical documents to more powerful models, ensuring a fast, efficient workflow.
